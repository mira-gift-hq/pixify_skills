# MasterPiece Clone

A powerful image-to-image and prompt-to-text workflow that clones and enhances masterpieces.

## ✨ Features

- Multi-image input support
- Prompt-controlled image transformation
- High-quality output generation
- Async workflow execution

## 🚀 How It Works

1. Provide two image URLs as inputs
2. Input a text prompt for guidance
3. AI analyzes and processes the images
4. Resulting masterpiece is generated

## 🧪 Example

### Input
```json
{
  "Image Input": "https://example.com/image1.jpg",
  "Image Input 1": "https://example.com/image2.jpg",
  "Text Input": "a cinematic portrait"
}
```
